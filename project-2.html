Credit Card Fraud Detection

Overview: Developed a LightGBM model to detect fraudulent credit card transactions with high precision and recall.

Key Features:

  • Preprocessed data by handling missing values, encoding features, and applying SMOTE for class balance.
  • Optimized LightGBM model using grid search and early stopping.
  • Visualized feature importance and fraud patterns using matplotlib and seaborn.
  • Evaluated performance with ROC-AUC and stratified k-fold cross-validation.

Tech Stack: Next.js, Python, LightGBM

Impact: Improved fraud detection accuracy by 15% compared to baseline models.
